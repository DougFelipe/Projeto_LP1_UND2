O projeto consiste em um gerenciador dos livros de uma biblioteca, onde se pode cadastrar um livro, alterar, remover e buscar. Armazena quem pegou o livro para ler, quais os autores que se tem no acervo da biblioteca, quais editoras se tem parceria.\hypertarget{md__c___projetos__b_t_i__projeto__l_p1__u_n_d2__r_e_a_d_m_e_autotoc_md1}{}\doxysubsection{Detalhes tecnicos sobre o projeto}\label{md__c___projetos__b_t_i__projeto__l_p1__u_n_d2__r_e_a_d_m_e_autotoc_md1}

\item Autores ultilizaremos o conceito de pilhas para manipular suas informações;

Será feita uma busca binaria quando for preciso fazer uma consulta nos livros, \mbox{\hyperlink{class_autor}{Autor}}, \mbox{\hyperlink{class_editora}{Editora}} e \mbox{\hyperlink{class_leitor}{Leitor}}. Para isso ultilizaremos de templates genericos para ter uma melhor economia de codigo, evitando que fique verboso.

 Onde o \textquotesingle{}T\textquotesingle{} será a entidade que esta buscando (\mbox{\hyperlink{class_livro}{Livro}}, autor, leitor, etc...), assim evita que se crie um metodo de busca para cada classe.\hypertarget{md__c___projetos__b_t_i__projeto__l_p1__u_n_d2__r_e_a_d_m_e_autotoc_md4}{}\doxysubsubsection{Ordenação}\label{md__c___projetos__b_t_i__projeto__l_p1__u_n_d2__r_e_a_d_m_e_autotoc_md4}
Ordenaremos da seguinte maneira\+:
\begin{DoxyItemize}
\item Os livros poderão ser ordenados de forma alfabetica, por ordem de lançamento. Para isso ultilizaremos o algoritmo Bubble Sort para a ordenação;
\item Os autores serão ordenados de forma alfabetica, ou por ordem temporal. Ultilizaremos o Selection Sort;
\item \mbox{\hyperlink{class_editora}{Editora}} e \mbox{\hyperlink{class_leitor}{Leitor}} tambem usarão do Selectioon sort;
